📊 Manufacturing Performance Dashboard (Power BI)

📌 Overview
This project showcases a comprehensive Manufacturing Performance Dashboard built using Power BI. The dashboard is designed to monitor and analyze key operational metrics across production processes, enabling data-driven decision-making in a manufacturing environment.
It covers multiple aspects of production performance such as efficiency, cost control, quality, maintenance, and energy usage.

🎯 Objectives
Monitor key manufacturing KPIs in one centralized dashboard
Identify inefficiencies in production processes
Analyze cost drivers and reduce operational expenses
Improve productivity, yield, and equipment effectiveness
Support strategic and operational decision-making


🧩 Dashboard Structure
1. Executive Summary
KPI overview, Performance trends, Key insights and highlights
2. Production Performance
OEE breakdow, Productivity analysis, Downtime analysis
3. Yield Analysis
RM & PM Yield tracking, Waste and loss breakdown
4. Cost & Energy
Unit cost breakdown, Energy usage (Electricity, Gas, Coal)
5. Quality (QA)
Reject rate, Defect analysis, Complaint trends
6. SHE
Incident tracking, Safety metrics
7. Maintenance
Breakdown vs planned maintenance, Machine performance (MTBF & MTTR)
8. EPI
Efficiency performance tracking across multiple factors

⚙️ Tools & Technologies
Power BI (Data Visualization & Dashboard)
DAX (Data Analysis Expressions)
Excel / CSV (Data Source)

📈 Key Insights
This dashboard helps answer critical business questions such as:
Why is unit cost increasing?
Which production line has the highest inefficiency?
Where do the biggest losses occur?
Which machines contribute most to downtime?
How does energy usage impact cost?

🚀 Features
Drill-down and drill-through analysis
Trend and variance analysis
KPI monitoring with dynamic visuals
